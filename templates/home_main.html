Product Review Sentiment Analyzer

About the Project

The Product Review Sentiment Analyzer is a web application that allows users to analyze reviews from Reddit and Flipkart. It utilizes web scraping techniques to gather reviews and performs sentiment analysis to provide insights into customer opinions about various products.

Features

  • Scrape Reviews from Reddit and Flipkart
  • Perform Sentiment Analysis on Reviews
  • Display Analyzed Sentiments (Positive, Negative, Neutral)
  • Download Reviews as CSV
  • Subscription Plans for Enhanced Features
